# fastClick

大致是说，移动浏览器 会在 touchend 和 click 事件之间，等待 300 - 350 ms，判断用户是否会进行双击手势用以缩放文字。 fastClick 源码大部分都是用来解决 iOS 各个版本各种奇奇怪怪的 BUG。

解决方案:

<meta name="viewport" content="user-scalable=no" />
1

或者

html {
  touch-action: manipulation;
}

或则要兼容老系统使用fastClick
